# Simple-Calculator
A simple calculator built in Python to practice basics.
# Simple Calculator

This is a beginner friendly calculator and it performs basic arithmetic operations like addition, subtraction, multiplication, and division.  

# Features

- Takes two numbers as input from user
- Performs the operation which is choosen by the user.
  - Addition
  - Subtraction
  - Multiplication
  - Division
 and after that show the result.

# How it works

Run the Python file "Calculator.py" and give the input to get the answer you want.
